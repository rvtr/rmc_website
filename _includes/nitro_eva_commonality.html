Important Info:


Most Nitro EVA cartridges are compiled as Self Checker, and simply have save data to determine if they should run in a different mode. As such, most of the actual dumps and downloads will be on the Self Checker page. Different modes can be selected in the app with the menu option at Checker Menu --> Configuration --> 01-02 Test Mode. Each test mode has a corresponding number, as listed below:

  • 0000: Self Checker
  • 0001: Aging
  • 0002: Flicker Adjuster
  • 0003: LCD Unit Checker
  • 0004: Self Checker(8M)
  • 0005: TP Calibration
  • 0006: IPL2 Self Rewrite
  • 0007: TP Accuracy
  • 0008: Test TP Jig
  • 0009: Only For Test

There are three exceptions to this: MIC, F-WRITER, RF Test, and the prototype. These may have unique sets of modes, or are separate programs entirely.
