Flashing DS prototype firmware to real hardware

Install the DS prototype firmware to DS lites and phats. This guide carries some risk, so please be sure to follow everything carefully.


How to install iQue firmware on a non-iQue DS lite

Replace the DS lite's firmware chip to one with a bigger capacity, and then install iQue firmware. This is an extremely dangerous process if you do not have experience soldering.
