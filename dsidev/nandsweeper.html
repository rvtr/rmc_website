--- layout: devinfo app: NANDSweeper ---

NANDSweeper is a tool created to destroy and read the DSi's NAND/SD card. The app comes in two versions (NTRJ and NTRJ-Secure), with only NTRJ being bootable. It's unknown why this tool exists, though considering how it was found in the TwlToolsRED repository (used to develop the internal IPL), it may have been to wipe prototypes. NandSweeper is a fork of a less distructive program called SD/NAND Aging Checker. The functions in NandSweeper are shown below:

  • SWEEP: The SWEEP function allows users to set every byte in NAND or the SD as 00, FF, 55, AA, or incremental/decremental and random patterns. The location to start writing is set through the Input Transfer Sector.
  • DUMP: DUMP displays a hex dump of the NAND and SD. The Input Sector Address sets the location to start reading from, and dpad-up and dpad-down are used to scroll through the hex dump.
