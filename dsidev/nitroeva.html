Nitro EVA

Nitro EVA, commonly known at NTR Aging, is a program to test and diagnose issues with DS consoles right off the assembly line. Nitro EVA has been used to test 3 generations of DS consoles (DS phat, DS lite, 3DS), with some small modifications between each version. The following is a list of each dumped version along with the console it was made for:
- v1.0, v5.1, v6.0 (used to test the DS phat)
- v8.0, v9.0, v10.0 (used to test the DS lite)
- v11.0, v11.1 (used to test the 3DS)

Test list

This is a list of the tested that can be accessed from Nitro EVA's menu. The tests are categorized by which menu they are available in.
EVALUATION MENU:
- CPU, Unkown Purpose
- MIC,Microphone tests and sampler
- LCD, Displays colored bars to test top screen
- LCD-2, Displays images to test top screen
- ROM HEADER, Display cartridge header info
- KEY, Displays button input values
- EMI DEMO, Model and rendering tests (need to confirm)
- NTR CONFIG, Displays the current system settings
- 2D(MAIN), Displays images from list to top screen
- 2D(SUB), Displays images from list to bottom screen
- 2D(VC), Diplays images from 2D(MAIN/SUB) in sequential order
- PMIC, Tests power and backlight controls
- KINSOKU CHK, Graphical button tester
- KEY TEST, Displays button input values

TEST PROGRAM:
- SELF CHECKER, Run full test on system
- AGING, Run full test on system
- FLICKER ADJUSTER, Displays thin bars, which cause flickering, to see if screens need to be adjusted
- LCD UNIT CHECKER, Displays thin bars, which cause flickering, to see if screens need to be adjusted
- TP CALIBRATION, Calibrates touch screen
- TP ACCURACY, Tests touch screen calibration
- TEST TP JIG, Tests touch screen calibration
- IPL2 REWRITE, Rewrites or sets system settings

MISC MENU:
- Test Screen, Displays colored bars to test top screen
- Test Sound, Plays music and sound effects
- Swap Display, Swap bottom and top screens
- Test Button, Displays button input values
- Test Nvram, Reads and writes system settings
- Dump Eeprom, Displays a hex dump of the cartridge, and allows basic hex editing
- TP Linearity, Tests touch screen calibration

Additional info

- Version 1.0 does not appear to have a menu to select specific tests, and attempting to exit to the menu will just turn the console off.
- The Nitro EVA UI was used in many other test programs, such as Mic Tester.
