NTR BOARD is a tool used to test DS lites and phats in the factory. NTR BOARD includes the standard BOARD and FINAL tests that similar tools use, but also includes the ability to write to the firmware stored on the WiFi module (IPL2). This ROM can be difficult to run. If using mGBA, you need to go to Tools > Game overrides > Set save type to Flash 1Mb > Save & Close before you can get anything booting.

The BOARD mode is used to test a DS motherboard and its outputs. The specific tests are somewhat similar to Nitro EVA AGING, with tests like audio output, screens, 3D graphics and effects, as well as memory.

FINAL is to be run after the DS has been fully assembled. This stage starts by working on the WiFi module. Language, birthdays and other basic settings are set and checked. Then speakers, headphones, and the sleep mode detection are checked.
