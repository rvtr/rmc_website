Even more 3DS factory tools

We just got three more 3DS factory tools from Kelly, DeadlyFoez (with data recovery help from Rairii), and an anonymous uploader. See the previous release for more of this software.

System Writing Program (BS_W)

BS_W is a tool to sign per-3DS files. This is a variant of BSL_W from the last release, and appears functionally the same, with added CheckRegion and CollectOptionalData functions. BS_W is run just before the firmware install process (CTR EVA IMPORT). The tool starts by connecting to a 3DS on a factory server. It reads, signs and then writes the SecureInfo, LocalFriendCodeSeed, and movable.sed in NAND. It then quits and allows IMPORT to install the firmware.

This version targets "JAN" consoles (n2DS XL)

In order to run the software, extract the BS_W folder and lineNumber.txt to the root of the C:\ drive. lineNumber.txt must contain a 4 digit number in plaintext and nothing more, otherwise BS_W will fail to boot. You may need to open and close the "debug" screen (top left button) before you can control the software.

Wireless and Audio (CR_WA)

CR_WA appears to be the same as FR_WA from the previous release, but instead has the addition of the Inspection_Result_CheckInspectionItems and Inspection_Result_RegistryInspectionItems functions, and it does not check the microphone stability (Inspection_Sound_CheckMicStability). The program works by connecting to a factory access point, testing the connection, disconnecting, then testing the microphone.

Both versions of this software target "KTR" consoles (n3DS)

In order to run the software, extract the CR_WA folder and lineNumber_CR_WA.txt to the root of the C:\ drive. lineNumber_CR_WA.txt must contain a 4 digit number in plaintext and nothing more, otherwise CR_WA will fail to boot. You may need to open and close the "debug" screen (top left button) before you can control the software.
