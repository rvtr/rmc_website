3DS factory tools

Today we've got seven 3DS factory tools from march42! All of these were run on factory PCs, hooked up to manufactured 3DSes through a cartridge adapter. This software has been recovered from factory SD cards, and will not work without the original factory environment. However, they will still boot and show us what the process of running them looked like. Should the original setups be replicated, dependencies for the software have also been included.
Table of contents
- System Writing Program
- FR Camera And Interface Software
- CTR Camera Sequencer Program
- FR Wireless And Audio Software

System Writing Program (BSL_W)

BSL_W is a tool to sign per-3DS files. It is run just before the firmware install process (CTR EVA IMPORT). The tool starts by connecting to a 3DS on a factory server. It reads, signs and then writes the SecureInfo, LocalFriendCodeSeed, and movable.sed in NAND. It then quits and allows IMPORT to install the firmware.

In order to run the software, extract the BSL_W folder and lineNumber.txt to the root of the C:\ drive. lineNumber.txt must contain a 4 digit number in plaintext and nothing more, otherwise BSL_W will fail to boot. You may need to open and close the "debug" screen (top left button) before you can control the software.

FR Camera And Interface Software (FR_CAM-IF)

FR_CAM-IF is a tool to test the camera. The 3DS will be made to take photos of color charts, where an operator will confirm that the camera has taken the photo correctly.

In order to run the software, extract the FR_CAM-IF folder and lineNumber_FR_CAM-IF.txt to the root of the C:\ drive. lineNumber_FR_CAM-IF.txt must contain a 4 digit number in plaintext and nothing more, otherwise FR_CAM-IF will fail to boot. You may need to open and close the "debug" screen (top left button) before you can control the software.

CTR Camera Sequencer Program

CTR Camera Sequencer Program is a tool used to test the 3DS cameras. A 3DS is connected to the Sequencer software by a cartridge's "HDMI" port, and is placed in a dark box. The 3DS is shown various images and Sequencer displays the camera output on the PC. The images shown are diagnostics such as color charts.

The first three photos below are from the 3DS factory setup manual. Pictured is the camera jig.

FR Wireless And Audio Software (FR_WA)

FR_WA is a tool to test the WiFi module and microphone. The 3DS will connect to a factory access point, test the connection, and repeat the process two more times. After this the microphone sensitivity is checked.

In order to run the software, extract the FR_WA folder and lineNumber_FR_WA.txt to the root of the C:\ drive. lineNumber_FR_WA.txt must contain a 4 digit number in plaintext and nothing more, otherwise FR_WA will fail to boot. You may need to open and close the "debug" screen (top left button) before you can control the software.
