System Writing Program (BSL_W)

BSL_W is a tool to sign per-3DS files. It is run just before the firmware install process (CTR EVA IMPORT). The tool starts by connecting to a 3DS on a factory server. It reads, signs and then writes the SecureInfo, LocalFriendCodeSeed, and movable.sed in NAND. It then quits and allows IMPORT to install the firmware.

In order to run the software, extract the BSL_W folder and lineNumber.txt to the root of the C:\ drive. lineNumber.txt must contain a 4 digit number in plaintext and nothing more, otherwise BSL_W will fail to boot. You may need to open and close the "debug" screen (top left button) before you can control the software.
