--- layout: devinfo app: SDMC_Launcher altname: SDMC Launcher category: [bootloader/ntrboot] sequence: FALSE group: true preview: /preview.png --- This is a special stage2 bootloader that boots an SRL off of the SD card. Only used in factories, this bootloader is installed on every DSi when it enters the BOARD phase of testing. This allows the DSi to run tools like PRE_IMPORT that will install an actual firmware. SDMC Launcher is found in most copies of TWL EVA and is treated as a fallback bootloader in case certain NAND writing operations fail.

Both the dev and prod versions are interchangable and will boot on either type of DSi. Normally stage2s will not boot at all if the signing is mixed, resulting in a stage1 error. However, SDMC Launcher's code checks the SCFG_OP register to make sure that the DSi signing matches SDMC Launcher's, so the stage2 will boot but immediately crash.
SRLs must be signed for the target console in order to be run.
