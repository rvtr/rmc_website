WM test is a test app included in all DS SDKs. It is used to test the Wireless Manager library. The WM library handles most local wireless functions like download play and local multiplayer. The tests from WM test are listed below:

  • Scan Test: For finding other DSes distributing software over NiFi and WiFi access points. WM Test allows you to narrow your search to specific channels
  • ScanEx Test: This is a more detailed version of Scan Test.
  • MeasureChannel Test: Checks if and how much a specific channel is being used,
  • Noise Mode: This will send noise over NiFi and jam legitimate signals

WM test has been embedded/forked into both the Nitro EVA prototype and RF Test.
